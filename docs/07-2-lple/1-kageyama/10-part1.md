**——收到了来自增田先生的给本作编曲的请求时，你有如何感受呢？**

景山：收到增田桑的委托，大概是一年前左右。能被委以用最新的音源和真实乐器去改编《宝可梦 红／绿》的乐曲，真是超高兴的。我离开 Game Freak 自立门户约 4 年半了（当时 2018 年 7 月），期间也积累了各种各样的经验，就一直想什么时候又可以给《宝可梦》做音乐呢。所以这次的任务很重，真的压力山大，但对我来说真是非常荣幸。

**——可以通俗易懂地介绍一下，本作的编曲是怎样的一项工作吗？**

简单来说就是将原本 Game Boy 时期由“3 音 + 噪声”制作的原曲，升级为使用各种乐器演奏的新乐曲。本作的编曲，我就都尽量去利用作为原曲根本的旋律、低音、对位旋律和节奏，调性和速度我也尽可能使之与原曲保持一致——就是这么一种尽量尊重原曲音响效果的方法。粉丝们都希望音乐“既能听着让人怀念，又能拥有全新的豪华配置”，所以不只是旋律，和弦进行、和声配置（和弦中音符的排列）这些所有原曲中的要素，都用最新的音源和乐器现场演奏录音来再现。

**——那么是怎样去构思改编这些 Game Boy 的电子音原曲呢？**

景山：首先是决定原曲的“3 音 + 噪声”都该由什么真实乐器去演奏。仔细去听原曲的话，其实是可以慢慢听出来它是带着什么乐器的印象而创作出来的。另外还要考虑不同乐器的音域，综合这些要素最终决定出要使用的乐器。现在的游戏机音轨数没有限制了，能够驾驭非常丰富的声音。这次我编曲的主要方法，就是先把作为骨架的 3 音轨认真搭好，然后在此基础上加入和音等之类的“血肉”。
