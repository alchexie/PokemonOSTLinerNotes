**——本作的编曲中作为“新的进化”的是哪一部分呢？**

景山：原曲因为当时 ROM 容量限制的关系长度都很短，但 Switch 就没有这个问题，所以现在可以收录长曲了。于是，曲子的第一个 loop 我会忠实于原曲，第二个 loop 我就会去改变乐器配置与和声，也就是说，第二 loop 可以实现更趣味、更自由、更豪华的编曲。我想这样既能满足原曲的粉丝，也能展现我自己的风格，希望大家一定要多听听，通过两个 loop 的不同编排，感受一下编曲中新的进化。

**——景山先生在离开 Game Freak 创立了自己的工作室后所获得的各种经验，是如何活用到本作中的？还有能告诉大家，与其他作品相比，给《宝可梦》系列制作音乐会有什么不一样呢？**

景山：另起炉灶之后，我在各种各样的作品中积累了很多录音的经验，当中包括海外和国内的完整管弦乐录制经验，这给本作的编曲带来了很大的帮助。然而要注意的一点是，给《宝可梦》制作音乐时，我会注意不要加入太多我自己的个人色彩进去。而最重要的，是要去延续增田和各位前辈一直以来所构筑的宝可梦音乐风格。在这一点上，我认为最关键的就是要把握好自己的原创分寸。
