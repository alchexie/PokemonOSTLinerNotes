- 增田顺一　总监／音乐
- 一之濑刚　音乐

**——《宝可梦 火红／叶绿》的音乐，是基于增田先生在《宝可梦 红／绿》时创作的曲子，由一之濑先生进行重新编曲制作的。首先请增田说一下，宝可梦的音乐究竟是如何创作出来的呢？**

增田：最开始创作主题曲的时候，我写了大约 30 个短乐句，现在的 **主题曲 _1.03 rg-1.01_** 就是从那里面来的，实际上那个旋律里是融入了宝可梦鸣叫的氛围。而当这首曲子完成时，我决定以这个旋律作为其他乐曲的主体方向。

**——那些令人印象深刻的战斗曲，您是以什么想法创作出来的？**

增田：我喜欢逐渐升温的压迫感，所以努力创造出这种激昂的感觉。战斗曲都是整得比较长的，目的是想尽量调动玩家的紧张感。

**——相反地，城镇的音乐让人听起来很舒心啊。**

增田：在这些地方，宝可梦是不会蹦出来的，所以我想通过旋律就能让人体会到“仅仅听音乐就可以感到安心”的感觉。以前 Game Boy 的画面是黑白的，道路和城镇很难区分，所以我致力于让玩家能够仅通过音乐就能了解当前所在位置是什么地方。

**——制作《火红／叶绿》的时候肯定要重新听当时的曲子，现在再听听又有何感受呢？**

增田：我感觉这些曲子并没有过时。

**——玩《火红／叶绿》的时候，音乐效果左右声道频繁切换，我真是太惊讶了！**

增田：这是一之濑的执着，

一之濑：我们听普通管弦乐时，声音是不会左右移动的对吧？但是这是游戏音乐，所以我往里加入了很多根据乐曲状况变化而蹦跶的“小玩意”。建议在玩《火红／叶绿》时最好带上耳机，会听到很多扬声器上听不到的声音哦。

**——请告诉我有这种“小玩意”的曲目吧。**

一之濑：首先是 **《圣安奴号》_1.36 rg-1.24_** 的汽笛。汽笛声并不是音效，而是与曲子是一体的，这完全是一个新的尝试，当时不知道增田会否同意我挺忐忑的。然后就是 **《最后之战 (VS劲敌)》_1.70 rg-1.43_**，听到增田给《红／绿》写的原曲时，我受到了很大的冲击，这次我想进一步增强这种冲击，并让玩家们吓一跳，于是就加入了类似人声合唱的声音。

**——七之岛的音乐您又是如何构思的呢？**

一之濑：增田为《金／银》写的曲子中，有我非常喜欢的曲目，我将这些重新编曲并用在 **七之岛 _1.59 1.63 1.64_** 上。猜猜都用了哪些曲子吧（笑）。

**——请告诉我你编得特别满意的曲子。**

一之濑：**《深灰市》_1.15 rg-1.09_**，这首我已经改编过3次了，分别是 Game Boy 的《宝可梦弹珠台》、《金／银》和这次的《火红／叶绿》。这一次我想是最忠实于原曲的，同时又充分加入了让人愉悦的和弦进行。虽然这首一般听着可能就过了，但我非常喜欢，这次改编得很不错，让我自卖自夸一下（笑）。

**——从 Game Boy 到 GBA，音乐制作的过程是否有很大变化？**

增田：要花费的时间变得多多了。因为可以通过采样来再现真实的声音，所以现在什么都能办到了，现实中不可能存在的声音也能弄出来。这次与《红宝石／蓝宝石》相比是全面有了提高，所以要将音乐容纳进游戏里是相当费劲。不削减曲目就放不进游戏的事态相当让人头痛，还好游戏的存储容量从 64MB 增加到了 128MB，算是帮我们解决了这个难题。

**——请解说一下几个附赠曲目。**

增田：代欧奇希斯是一种根据不同卡带而改变形态的宝可梦，所以我也通过音乐来表现这种形态改变。**《代欧奇希斯登场》_2.01_** 的曲调，就像是摇曳的海市蜃楼。而 **《对战 (VS代欧奇希斯)》_2.02_** 中，因为主角是宇宙来的宝可梦，要展示出宇宙无限的可能性，所以我将强有力的旋律放在上行音阶中。

一之濑：**《教教我！欧尼桑》_2.03_** 应该就像是真的有一个教育节目的大哥哥在那唱的感觉？而 **《memory P》_2.04_** 则是送给和宝可梦一起一路走过来的人们，请一边听歌一边唤醒你的回忆吧。

增田：**《火箭队基地》_2.05_** 让人觉得是在窥探宝可梦世界背面的刺耳曲调。**《宝可梦塔1997》_2.06_** 则使用了巴厘岛的乐器甘美兰和竹琴，通过缓慢的噪音去使人大脑放空。**《道馆馆主的力量》_2.07_** 原曲是以宇宙广阔的空间为印象创作的，这次则是将表现重点放在力量上。

**——请你们给将来想从事游戏音乐制作的朋友提一些建议。**

增田：不管主题是人、动物还是其他东西，试着根据其动态变化把曲子或简短几句哼出来，说不定是种不错的训练方法。

**——回顾一下《火红／叶绿》的工作，你们有什么感想呢？**

一之濑：工作都完成后，我问增田这次的改编怎么样，他给我说：“这次的改编很好！”我都高兴傻了，人家可是这些曲子的原作者啊。

增田：嗯！

一之濑：虽然是有点逼他这么说的（笑）。

增田：不要把赞美的话当成笑话啊喂！

（2004 年 3 月收录）
