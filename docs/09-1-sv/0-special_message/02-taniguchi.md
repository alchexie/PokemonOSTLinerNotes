**《战斗！天星队》_1.31 (31) - 戦闘！スター団_** →我制作的第一首曲子

最初我只是简单以“不知道要搞什么幺蛾子的团伙”这个要旨来开始制作这个曲子的，但随着对天星队成立经过和背景的理解加深，我对当初这个“邪恶组织”的印象也逐渐发生了改变。

我一度提出我的看法并得到确认后，我对这曲子做了大幅的改动。起初，它的曲调更为轻快。

**《战斗！天星队老大》_2.37 (87) - 戦闘！スター団ボス_** →这车是……宝可梦？

（天星队车）第一眼给我的冲击太大了，记得当时因此而导致作曲迟迟不能推进，整得我一个劲地来回踱步。到现在，我仍然没办法冷静地去听这首曲子。

当我看到毅然站在这异型般的天星队车上的老大时，我觉得，一个人那样站着真是太帅了。
另外在这首曲子中，森空青先生的吉他是必不可少的，他的演奏毫无疑问充满了热情，我在此非常感谢他。
