**A1.**

增田指出，BW 音乐的方向是“融入世界的音乐”，主要理由是因为合众地区是一个异文化相互交融的地方。为了将这种印象具象化，我们声音组对世界音乐进行了研究，从结果来看我们是成功将研究成果反映到作品里去了。

但比起这种研究，我认为更重要的是我们在领队景山的带领下，迎来了新成员足立。新旧成员的个性碰撞，是这次能制作出与异文化融合的合众地区相衬的音乐的最大原因。

然鹅……说到个性自己就一直还是那个鸟样啊！杯具杯具杯具杯具、等离子大杯具！

还要说的是，是优秀的程序员们实现了景山极好的提案，使得比起过去的宝可梦游戏，音质和乐器多样性都有了提升。拜此所赐，我们庆幸有了比以往更加自由的作曲环境，大家要是能从 CD 中感受到这种变化，那我就开心得要燃起来辣！
