title: '景山将太'
---

**A1.**

自己第一次当音响监督的作品就是完全新作的宝可梦 BW，说自己没有压力山大那是骗人的……但比起我想创造出让宝可梦老粉和新玩家朋友都能感动的作品的心情，压力可谓微不足道，总之开发过程非常愉快和充实。
增田总监说，希望 BW 不受限于已有的框架且要去追求崭新的“宝可梦之声”，所以我们集思广益，一起探讨要如何通过声音来给大家带来惊喜。
首先，合众地区是一个人们互相包容彼此不同的文化、价值观和思维方式而共同生活的地方，为了让曲子也能表现这种世界观，我们往其中加入了各种各样的世界音乐要素。

然后，这作的 BGM 里加入了根据玩家状况或行动的[交互变化](https://www.serebii.net/blackwhite/bgmusic.shtml)。道路、城镇、迷宫、战斗、季节变化等等……在此之中藏着各种各样的声音趣味，请大家务必找出来啊！

其他方面，我们在音质上、剧情与音乐演出的契合度上都是由始至终地一丝不苟。声音团队的大家对 BW 强烈的执着和爱是多得说不尽的。我为大家的共同成果感到自豪，衷心希望这能够成为可以触动大家心灵的声音。

**A2.**

莱西拉姆、捷克罗姆的剧情曲 **《N 的龙》与《龙觉醒》_2.34 2.35_**，是 N 的龙声势浩大登场时的力量和狂野（动）、与另一只龙在主角眼前从石头中复苏的神秘感（静）的对比。为了突出这种反差，我努力利用 DS 有限的音色去表现壮大厚重的管弦乐效果。

**A3.**

我认为这次的作品，是 Game Freak 声音团队里大家不同的个性互相碰撞、又互相融合、最终化为一体的成果。请大家通过这张 CD，尽情享受这完成了更大进化的宝可梦之声吧。
