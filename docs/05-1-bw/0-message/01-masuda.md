**A1.**

创造一个作品是需要耗费巨大的精力的，这次我也将我想做的东西全部都做出来了！所以我已经没电了，感觉已被榨干（苦笑）。
BW 是我们决定以全新的理念制作的，所以我们开发者首先就一定不能被宝可梦已有的概念所束缚。音乐方面，我们也有专门抽时间来讨论要如何才能构筑和以往不一样的音乐风格。

**A2.**

请把 **城镇的音乐*1.06 1.17 1.30 1.41 1.51 1.55 2.02 2.08 2.13 2.23 2.24 3.30 3.47 3.48 4.04*** 连起来听听吧，每一首都有各自不同的特色，我想你会觉得像是到了异国一样。这就是我所希望的通过音乐来表现合众地区的多样化。
听完觉得怎么样呢？不觉得我们团队很完美地做到了这一点吗？

**A3.**

我认为游戏音乐的前提在于游戏。我们设计音乐就是希望玩家能感受到操作时的愉悦和情绪的起伏。请大家回想自己游戏时的情感和行动，同时听一下音乐吧，说不定你会有新的发现！
