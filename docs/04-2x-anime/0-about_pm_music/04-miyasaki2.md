**——另一方面，宫崎先生将制作方法截然不同的游戏音乐改编成管弦乐演奏的动画音乐，这其中最困难的地方是什么呢？**

宫崎：《宝可梦》早期的音乐，能同时演奏的音轨很少对吧。

增田：是的，《红／绿》的时候，能同时发声的就只有 3 条轨外加一个噪声通道。

宫崎：所以，这些曲子中包含着实际上没有演奏出来的音符，但这些音符在增田桑的脑海里应该又是响着的。那这些声音在增田的脑海里是怎么样奏响的呢？如果和我想的完全不一样，那怎么办？这就是最难的地方了。毕竟最初的改编会决定总体的方向，我当时是非常担心这个问题，老是想着“这么整真的可以吗？”

增田：这么说，我觉得宫崎桑非常厉害，真的就是将我脑内的音符全部拣出来了。

宫崎：经常在电影中使用的 **序曲 _1.01 1.02 1.03 rg-1.01_**，它是我心中之最。曲子本身就很好，我想如果没有它，大概也就没有《宝可梦》音乐的今天吧。

**——原来如此原来如此。**

宫崎：那么说来在 Game Freak，在决定游戏世界观的过程中，像“曲子虽好但印象不符”之类的判断，是由增田桑来完成吗？

增田：我现在作为游戏总监，确实会要干这种事，如果说在意的话，那这确实是我非常在意的部分。比如说要制作一张 CD，如果全部都是重摇滚，果然听下去会很累吧，所以当中需要好好放松一下。因此根据游戏中冒险的顺序，大致构建好音乐的走向，这点我是很留心的。

一之濑：增田特别注重一些东西——比如，关于剧本上或世界观中重要的地方，他会跟我们说清楚该有的印象。除此之外的地方，我们要是试着加入自己的想法，如果增田啥都不说，那就说明是 OK 的。

增田：我通常不怎么会给否定意见。但这种我会说，比如，城都地区虽说是以京都附近一带的街区作为原型，但是也不能太强调“和风”。我想通过“和风”地点与“非和”音乐的配合，创作出更加不一样的世界观。
