**——机会难得，有请增田先生说说。**

增田：我啊，大概是小学三、四年级的时候吧。当时家里有《桂河大桥》和《宾虚》的原声带，我记得我听着这些音乐，然后用竖笛模仿之，这是我第一次觉得音乐很有趣的瞬间。还有就是六年级的时候听到的 YMO（Yellow Magic Orchestra），从那时起，我就迷上电子乐了，爱听 Tangerine Dream、Ultravox、Kraftwerk 之类的。我让父母给我买了一台收录机，结果就是只用来听摇滚和西洋乐了，所以在学校里几乎就没有聊得来的人（笑）。而且唱片很贵啊，买不起买不起。

**——所以就将广播里喜欢的音乐录下来是吧（笑）？**

增田：是啊，不过就算歌曲放出来，我也不知道这是什么歌、是谁唱的……想要知道这些信息，就得去买杂志来看，那个年代就是这样的。

**——那增田先生从事现在音乐活动的源头大概又追溯到什么时候呢？**

增田：好像是中学入学的时候吧？那时候我又求父母给买了一台 KROG MS-10 合成器，然后我一个朋友也喜欢这些玩意，我俩还用两台磁带录音机一起搞多重录音。所以，我并没有专门学过音乐。

**——是这样啊！**

增田：高中的时候，我在吹奏乐团里吹长号，也是没有专门学过的。基本上我就喜欢电子和摇滚，这些与宝可梦世界相去甚远，更不用说 Marilyn Manson 和 The Spotnicks 那种了。

一之濑：这个啊，增田桑的喜好在战斗曲里很明显呢（笑）。

**——那一之濑先生也说说你的？**

一之濑：我幼儿园的时候，被送去游泳学校和音乐教室。然后游泳学校我是哭着不要去，只有音乐教室是坚持下来了。但是，我是完全不写作业，老师就跟我说“不写不能让你升到上面的班里去！”

**——啊哈哈（笑）!**

一之濑：之后，我转学钢琴，但又只练习自己喜欢的曲子。接着是初中的时候吧，发现自己果然是最喜欢肖邦了。有了“好想弹这首啊”的想法，于是我就拼命去练习那些难度大的曲子。也是这时候，我迷上了游戏音乐，像 FC 游戏里的音乐，又或是游戏中心里的那些。我把录音器材带去游戏中心，把那的音乐录下来，然后回去用钢琴弹出来。

**——听你这么说，感觉你就是注定要加入 Game Freak 的。**

一之濑：其实那个时候，我也没想过要从事音乐，加入 Game Freak 也只是希望能掌握一技之长而已。那时候是偶尔看见了兼职招聘信息，就请求来这里学习一下。然后我开始学编程，但没什么长进，不知不觉间又搞起音乐来了（笑）。另外，我想起了我开始从事音乐的契机。记得是以前搬家的时候，我在家中发现了胎教用的唱片，这似乎是我还在妈妈肚子里时听的音乐。看到那些曲目，发现全都是我自己弹钢琴时的选曲。

佐藤：是肖邦吗？

一之濑：嗯，肖邦和李斯特，所以看来胎教是真有效果啊（笑）。

佐藤：我也有类似的经历。听说我还没到1岁的时候，爸爸就一直给我放约德尔音乐的唱片，然后有一天，这时候我都还没学会说话呢，就突然站在被子上唱起来了。然后父母就对我抱有了“这娃说不定有音乐才能”的希望，然后从3岁到高二，我都一直有学习钢琴。还有就是，我读的是基督学校，对赞美诗是倍感亲切，所以十七八岁之前，我都只听赞美诗和钢琴曲。

**——那你是怎么加入 Game Freak 的？**

佐藤：我是偶尔在招聘杂志上看到 Game Freak 在招聘策划，之后我在这个职位工作了七八年，然后偶然听说声音组那边有个人辞职了。在此之前我就提过我会弹钢琴的，这次被问到“要不要试试？”，那我就试试吧。

一之濑：哎呀，佐藤的耳朵真灵啊。
