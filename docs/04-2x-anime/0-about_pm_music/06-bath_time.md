增田：我最感兴趣的是，宫崎桑是怎么编曲作业的？是直接往乐谱上写吗？

宫崎：这十几年变化挺大的，最近是，在管弦乐现场演奏录音之前，我会用合成器等等去模拟音色然后录入到 DEMO 中去确认全部乐曲的印象。而我最初干这个的时候，一开始是没有乐谱的，所以暂且只能用耳朵将听到的大概“抄”下来，然后拿这个抄下来的谱子和音响监督给我的菜单进行对照，从而逐步去构建曲子的印象。

增田：这个谱子，是按照每种乐器的音域来写的吗？不同乐器是纵向排列出来？还是一个乐器记一个谱呢？

宫崎：两种都有。如果是旋律印象深刻的曲子，那就先抄旋律，然后再去考虑合适的和弦。相反如果是以节奏为中心的曲子，那就先做好节奏，然后再边弹边想后面的。

增田：那这么说，音色或许不预先确定会更好。我自己的话，就想在游戏里使用一些没听过的声音。所以我使用音色的方法稍微有些不一样，但交给宫崎桑的话，看是能改编成不一样的东西。

**——宫崎先生有没有什么是想问 Game Freak 的各位的？**

宫崎：大家会在什么时候冒出作曲灵感呢？我是坐在桌子前的时候、泡澡的时候，还有吃饭的时候……

增田：这点我想我大概也是一样的，洗澡或乘坐电车的时候，会突然想到一些东西，然后就一边口哨吹出来一边手机录下来，过一会再重新听一遍。但有时也会意外地录了一些完全用不了的东西进去（笑）。

一之濑：我果然和宫崎桑一样，也是淋浴洗澡的时候旋律构想就出来了。或许被工作截止日期追杀又突然解放的时候，就是我最能想到曲子的时刻。这话太社畜了抱歉（笑）。

佐藤：我相反，我不对着桌面就憋不出来东西，然后如果你跟我说“木有时间啦，只剩一个钟啦！”，我就马上能整出来。

**——越是抱佛脚越有用是吧？**

佐藤：是啊。旋律想不出来，我就从低音和装饰音开始写。首先将想用到的装饰音一下子堆进去，在只剩一小时的时候又把之前没用过的音加进去，最后应该就长得像旋律了（笑）。

**——感觉是先画好的草图然后再填东西进去的样子。**

佐藤：是滴。

增田：原来是往后才考虑旋律的流派啊。

佐藤：没想到吧。

增田：我是优先考虑旋律的。

一之濑：+1。

宫崎：同上，我也是从旋律开始。

佐藤：原来我没有朋友啊（笑）。
