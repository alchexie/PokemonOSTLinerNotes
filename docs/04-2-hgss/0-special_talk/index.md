- 一之濑刚　音响监督
- 景山将太　声音
- 佐藤仁美　声音
- 桥田拓人　声音

**——首先请各位自我介绍和说一下自己负责的部分吧。**

一之濑：我是声音组的领队，除了作曲、编曲，也负责统筹全局。

景山：游戏内的音源制作、BGM 和 ME 的编排以及新曲的创作，还有就是主要负责 Game Boy 音乐的再现。

佐藤：将从《白金》继承到《心金／魂银》的设施等相关曲目进行移植，也担当狩猎地带等地的新曲。

桥田：我主要是给关都地区的音乐编曲。

增田：《金／银》的原曲是我和一之濑一起搞的。这一次我是作为制作人对全体工作进行监修。

**——首先让我吃惊的是音乐团队有 5 个人了，为什么这次的阵容会如此庞大呢？**

一之濑：早在企画阶段，新的设施、新的剧情、还有 GB Player 等要素就不断加进来了，所以曲目数量也一下子上去了。SE（音效）也是一个原因，江河、海洋等的环境音在制作上非常讲究，声音相关的工作量就变得惊人的庞大。

增田：原作《金／银》的曲目数量本来就很多，正因如此，大家能非常高水平、高质量地完成如此大量的音乐，我深感佩服。

**——能告诉我这次为《心金／魂银》新作的曲子吗？**

景山：这次的一个新要素是 **“宝可梦全能竞技赛”_2.40 2.41 2.42 2.43 2.44 2.45 2.46 2.47 2.48 2.49_**，我不希望大家太觉得这是一个普通的运动会，所以我当目标听众是提高了一个年龄段那样去创作的。实际上比赛开始时“3、2、1、开始”的倒数与音乐是同步的，我非常注重游戏起步时与音乐的配合，我想这样一来玩家的游戏体验也会更好。

佐藤：我负责的新曲是 **《47号道路》和《狩猎地带》_2.31 2.32 2.33_**。47 号道路是充满大自然气息的地方，曲子也是宽宏的，让人紧张又兴奋地想着“接下来等待我的会是什么样的地方呢？”结果到达了狩猎地带一看，“啊咧？意外很大众化的地方啊！”我就希望通过音乐表达这种反差。另外《狩猎地带》我也试着填了词的，请大家试试跟着在狩猎地带入口处的男孩子一起唱一下吧。

> 《心金／魂银》原声 CD 的盒子内，有一张“早期特典”，自发售日起到2010年3月31日，玩家可以发空邮件给官方指定邮箱，获得两首附赠曲目的下载链接。附赠曲目为景山将太cos巴奥巴园长演唱的《狩猎地带园歌》及其伴奏。到今天，人声版还可以在互联网上找到，但伴奏版却是毫无踪迹。

一之濑：一开始，**洛奇亚的战斗曲 _2.75_** 我是脑补不能……这时候看到增田来到我边上，我就让他告诉我洛奇亚曲子的印象是怎么样的。然后增田玩笑似的给我弹了三四个音，我听了之后秒懂，“原来是这种感觉啊！”接着就是异常顺畅得写完了这首曲子。然后最让我困扰的是 **凤王的战斗曲 _1.87_**，凤王在我心中是有一种“和风”的印象在的，但如果和风过于明显的的话，又会与宝可梦的世界格格不入，这让我非常烦恼。为了能够表现凤王的力量与庄严以及其不死鸟般的形象，直到截止日期之前，这曲子我都一直在反复修改又划掉又修改，最后算是完成了，但感觉还是像和风了。

**——《金／银》BGM 的再编曲是基于什么样的想法制作的呢？**

一之濑：我原本就是《金／银》的作曲者，这次在 DS 平台上我是想尝试在 Game Boy 上不能实现的编曲方式，然而老玩家也期待新的音乐中会保留以往音乐的氛围，这点我也不想让他们失望。但换句话说，在成品质量和不让大家失望这个平衡之间我是尽力了，要是你真的失望了我也只能说声对不起了（笑）。

景山：我也是，非常热切想回应过去乐在《金／银》之中的各位的期待。在 Game Boy 上，尽管只有“3 音”，但曲子的表现力非常强，我希望我可以尽可能发挥出原曲的优点。再者在这次《心金／魂银》中，我在努力提高音质的同时，也非常注重声音的透明度与美感。如果大家能感受到这种意义上的进化我会很高兴的。

> **3 音**：早期由于技术局限，Game Boy 内置音源能够同时发声的只有 3 个音高通道 + 1 个噪声通道（通俗说，3 个音高通道用于旋律，噪声通道用于打击乐）。3个音高通道对应不同的波形/音色，所以最多只有三条音轨，当时的游戏音乐就是通过对此的灵活运用、组合创作出来的。

佐藤：Game Boy 版的音乐是由“3 音”构成的，所以记住原曲主旋律的人会比较多。但我想可能也有人会非常喜欢原有的副旋律和装饰音的，所以我编曲的时候也尽力保留原来“3 音”所展现的内容。

桥田：如何将原曲中的细节置换到新音源，是我最着重考虑的事情。虽然原曲仅由“3 音”完成、仅由“3 音”奏响，但我想其中有一些曲子，作者在创作它们的时候脑内回响的肯定是不一样的声音。这一次，我就像是要为仅有“3 音”的画卷填上色彩一样，为这种能够想象其他声音在其中的曲子，添加新的音色并重现。

**——对于老粉来说，这张 CD 第三张盘收录的 GB Player 的内容是个话题呢。**

一之濑：原来 Game Boy 版的《金／银》是没有发行原声 CD 的，在考虑“玩家会喜欢的东西”时，就希望能把可以听到老曲的 GB Player 加入到企画中。在游戏里，GB Player 的解锁条件是集齐 16 个道馆徽章，这是给努力的玩家们的一个奖励。对于没听说过 Game Boy 版金银的人来说，“3 音”的原曲说不定听着也会有新鲜感。

景山：基本上这就是为了情怀杀而制作的，但实际上也有一些新曲是按照 Game Boy 音乐的样子去制作的，比如宝可梦全能竞技赛等等的《心金／魂银》中的一部分新曲、**《金／银》中没有重编的《红／绿》中的城镇曲 _3.65 3.81_**，这次都收录了。请大家务必再次游历一次本作的地图，再听一遍 Game Boy 风的音乐吧！

另外这些 Game Boy 音乐，并不是直接从《金／银》移植过来的，而是这次用 DS 的音源从 1 开始重新制作的。我认为成品已经是相当忠实地还原了原曲了，要是大家都喜欢就好了。

**——那么请大家从各自负责的曲目中推荐一些吧，顺便也说一下理由。**

增田：将阿尔宙斯带到《心金／魂银》中才会首次听到的 **阿尔宙斯的曲子 _2.84_**。为了表现这位超越一切的神，所以这听起来是会有一点沉重和可怕的。

一之濑：重新编曲的有 **《自然公园》_1.58 gs-3.37_**，为了这首曲子，我拜托了景山帮忙在其中加入了钢琴这种比其他乐器音色都更占空间的奢侈品。希望大家在可以听到美妙钢琴声的公园里，能够享受这治愈般的片刻。接着就是 **神都遗迹 _2.83_** 的曲子，这是天冠山的改编，后半段也混入了其他宝可梦的主题，请大家尝试找出来吧。

景山：我推荐 **博士助手（阿响和琴音）的曲子 _1.06 2.15_**。在我擅自的想象中，主人公对助手是带着初恋的情感的，所以我就想把这首曲子写成那种会让人心动的感觉，不过看来效果是挺好的。接着就是 **雷公、炎帝、水君的战斗曲 _1.68 1.83 2.13_**，一之濑强烈要求编曲时要强调它们各自的属性，我想粉丝们对这一点应该也会感到满意的。

桥田：在我所负责的为 GB Player 编曲的新曲中，有一首叫做 **《狩猎地带入口》_3.83 2.32_**，这是一首打击乐非常活跃的曲子，如果“3 音”的版本去掉打击乐器，那曲子原有的印象就会被破坏。所以，我是下了各种创意工夫去再现“3 音”的打击乐的，希望大家都去听一听。

佐藤：我推荐 **《冰雪小径》_1.79 gs-3.55_**，从《金／银》开始，这首就是我最喜欢的曲子了。正因如此，我编曲时尽量不去改动原曲，而是在此基础上增加一些凉飕飕的氛围。还有一首就是 **《Ending》_2.80 gs-3.99_**。因为不集齐关都地区的 8 个徽章就拿不到 GB Player，从而听不到那种经典的电子音，所以通关结尾曲的主旋律里我就使用了类似 Game Boy 的那种音色。我希望普通玩家在这里可以有“啊！这真让人怀念啊”的感觉。

**——最后，想请制作人增田先生谈谈对《心金／魂银》音乐的感想。**

增田：这次的作品完成得非常讲究，编曲工作也是以制作新曲一样的势头来完成的。整张CD可谓装满了值得听的成果，听完会有如饱餐一顿，总之无论哪首都是良曲！

（2009 年 7 月 30 日收录）
