- 黑天英明（硕士研究生毕业后从事音乐制作，除了游戏也参与流行音乐与电影等音乐制作领域）

接下来为大家送上来自深度参与《宝可梦 欧米伽红宝石／阿尔法蓝宝石》声音制作的音乐人们的热情寄语。

**景山将太：《宝可梦 ΩR·αS》音乐的主旨是怀旧与创新共存**

在《欧米伽红宝石／阿尔法蓝宝石》开发初期，我以音响监督的身份，与总监大森一起就音乐的方向性商讨过，最后决定本作音乐的主旨是“怀旧与创新共存”。借助 3DS 的流式音源，本作音乐的表现形式也可谓大幅升级，原曲也就能进行更豪华的改编。我想，如果当时没有声音通道数量和音色的限制，一定也会发展成现在这种编曲风格吧。这么想着，就最大限度地去尊重原曲的优点，然后细心地去改编。编曲过程中，我也会向增田和一之濑请教，问他们当时是以怎样的想法去制作这些曲子的。项目途中我离开了 Game Freak 自立门户，但虽然如此，我还是希望能为这个项目负责到底，所以最终以音响经理的身份，和大家一起完成制作。

至于问我喜欢哪首曲子那真是列举不完，但我把在本作中想要表达的全部都浓缩到了 **《共存》_3.15_** 这首原作没有的曲子中。海洋与陆地、人类与宝可梦、都与自然共生。想象大地再度充满自然生机，怀着对自然的敬畏之情，我创作了这首庄严、宽宏又优美的乐曲

**黑田英明：从作曲与编曲，到音效、宝可梦叫声，我参与了很多**

这是我继《宝可梦 X／Y》后的再次参与。这次除了音效和叫声制作、乐器演奏之外，我还有一部分乐曲的作曲和编曲工作。和《X／Y》不一样，这一次是复刻作品，因为我没玩过原作的《红宝石／蓝宝石》，所以首先得要把这游戏玩一遍。在此基础上，制作音乐过程中也有和大森总监一起就“要多大程度忠实于原曲”、“新要素要加入多少”这些问题进行了充分的讨论。

> 黑田在《X／Y》中没有参加作曲与编曲，而是作为吉他手参与。

本作中我亲自作曲的并不多，但我特别喜欢 **《per aspera ad astra!》_4.07_** 这一首。这首曲子是在主人公为破坏逼近丰缘的陨石而乘上烈空坐起飞的场景中播放的，大森先生希望这个场景不要过于沉重，而是要传达出来“现在要去拯救世界了！”这种积极信息，所以我就交出来这么一个富有戏剧性张力、又能表现决心的曲调。而破坏陨石也是希嘉娜的夙愿，于是为了可以体现和捎上她的心意，我在曲中好几处都用上了改编自她主题曲 **《悠久的继𠄘者们》_4.01_** 中的乐句。希望大家听的时候能够留意这些细节。
