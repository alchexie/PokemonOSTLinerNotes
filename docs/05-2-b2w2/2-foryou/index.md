**增田顺一**

豪华的 CD 完成了！是不是很棒呢？描绘《宝可梦 黑／白》两年后的世界的《宝可梦 黑2／白2》，当中的音乐也一样，经历了两年的变化，最终进化了！一之濑和佐藤两个人感受的碰撞与交融，然后“共鸣”，我认为这是一张很有深度的音乐作品。

然后，至今未收录的《绿宝石》和《白金》的音乐这次也终于能够带给大家了！《绿宝石》是 2004 年发售的，现在去听大概会觉得是怀旧了（笑）。请大家尽情欣赏这展现不同广阔魅力的宝可梦音乐吧！

**一之濑刚**

《绿宝石》中 **对战开拓区的头脑对战 _e-4.15_**，是我为宝可梦系列写的第一首战斗曲。紧接着前奏的地方有一组奇特的低音乐句，我记得我当时是根据对战开始前双方训练家平移出场互相对视的画面来脑补创作的。希望有一天自己能亲自重编这首曲子！也希望宝可梦的音乐今后能不断进化，请多指教呢！

**佐藤仁美**

在《白金》中，为了契合骑拉帝纳的形象，在 **标题曲和相关剧情曲目 _pt-4.18 pt-4.19 pt-4.27 pt-4.28 pt-4.29_** 中都加入了共通的、类似嘶鸣声的刺耳音效，听着就像是原本没有交集的两个世界彼此靠近之后产生的那种扭曲感……

至于其他的曲子，因为主线剧情的整体氛围是比较沉重的，所以我尽情发挥了自己的喜好，创作了一些欢快明亮的曲子。

**景山将太**

非常高兴这次能够将包括未收录曲目的音乐带给大家！尤其是 **《决胜！宝可梦世界锦标赛》_3.36_**，这是我全力为经历战斗脱颖而出的勇者们创作的决赛 BGM！在“2”里，宝可梦音乐又有了进一步的进化，请大家尽情享受！

**足立美奈子**

**《宝可梦派对！》_3.33_** 原本是为电视节目《Pokémon Smash》的联动企划创作的，我当时是感受到了各位节目主持人的阳光与元气而写下这首曲子！希望它能与《宝可梦 黑2／白2》的众多优秀曲子一起，受到大家的喜爱。
