《[宝可梦 THE ORIGIN](https://wiki.52poke.com/wiki/%E5%AE%9D%E5%8F%AF%E6%A2%A6_THE_ORIGIN)》剧中配乐里，有 5 首是由 Game Freak 的景山将太先生改编的，本 CD 也特别将这 5 首乐曲作为 **附赠曲目 _4.78 4.80 4.82 4.84 4.86_** 收录（曲目 4.78、4.80、4.82、4.84、4.86）。同时这些曲子的 **原曲 _rg-1.01 rg-1.07 rg-1.15 rg-1.43 rg-1.30_**（曲目 4.77、4.79、4.81、4.83、4.85）也收录于此，希望大家对比听听，感受一下宝可梦音乐的进化。

**景山将太：发挥原曲的优点构筑新的宝可梦音乐**

这次我是第一次给《宝可梦》动画配乐，真的非常荣幸。为了能让曾玩过《宝可梦 红／绿》的大家同时感受怀旧与新奇，我将原曲的优点最大限度地发挥出来，精心进行了这次改编。这回的音乐概念是基于“为宝可梦动画指示新方向”这一想法，在电子舞曲的节奏上加入真实乐器演奏来进行编排。我想，把《宝可梦 红／绿》的音源混进来，也是宝可梦音乐新的表现方式！
