**——我觉得《宝可梦 红／绿》（以下简称《红／绿》）的乐曲，大多数都是平易近人、朗朗上口的，在开发当时，作曲和编曲方面，您都特别注意了什么呢？**

增田：创作《红／绿》曲子的时候，我特别注重融合不同风格的音乐，同时把游戏音乐“易懂”这一特征作为重点。我的做法就是先写出抓耳的旋律，然后才加上低音与和弦（和音）。因为 PSG（Game Boy 内置音源“Programmable Sound Generator (可编程声音发生器)”的简称）能够同时发声的只有 3 个音高通道 + 1 个噪声通道，所以我一开始就没打算从和弦来入手作曲。

> 在 Game Boy 上，PSG 由 4 个声道组成：3 个有音高的通道 + 1 个噪声通道，其中两个方波通道负责旋律和和声，一个波形通道常用于低音或特殊音色，最后的噪声通道用于鼓点和音效；它们一起通过简单参数实时合成出经典的 8-bit 游戏音乐。

**——认真反复去听，我能够感受到游戏音乐特有的“畅快感”。比如 **《常青森林》_1.12_**，不断重复同一个乐句感觉还挺上头。**

增田：我非常在意如何去调动玩家的情绪，比如一首转调比较多的曲子，在回到本来的调时，我会让音乐暂且安静一下，大家到这就会觉得“好了，要归位了”。而低音通常负责维持节奏，即使是其他声音通道播放音效的时候，也尽可能保持整体节奏不被破坏。然后，为了节省有限的 ROM 容量，也会有使用 repeat（乐句反复）来处理。也许就是这些功夫，造就了这些符合当时那个年代的曲调。其实，《常青森林》是将 MIDI（用电脑制作的演奏数据）转换成 Game Boy 使用的格式之后，再手动微调参数作成的。而低音部分的循环我则是故意错开一小节，而且听得越久这种错位也越大，这也算是我花的一个心思。

**——听起来还藏着不少其他各种各样的小心思啊！**

增田：从开发当时到现在，已经 20 多年过去了，我自己也想不起来全部了，不过在《红／绿》时期，确实应该有不少的。比如《红／绿》和《蓝》、《皮卡丘》的存档音效，就是有些不一样的。然后就是使用传输线通讯对战的时候，两台 Game Boy 的音乐会有细微的不同步，这样的话对战时把音量调大，两台 Game Boy 播放的音乐就会像立体声一样，听起来会更爽快呢。
