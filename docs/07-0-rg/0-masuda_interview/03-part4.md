---

**——制作叫声基础音的时候，是如何展开想象进行设计的呢？**

增田：是从宝可梦的生态开始考虑的，比如发声器官的构造、体型大小、习性、是否强壮、是否温顺、是否敏捷等等。超梦的“嘟噜噜噜”就是通过把音高逐步调高来表现其强大，迷唇姐很有特色的“叮叮叮嘀叮叮”则是为了突出它的性格。另外，如果只考虑宝可梦的外形，那就会被视觉形象所影响，做出来的叫声确实会很像回事，但也会少了些意外性。我认为，叫声有着给视觉形象做补充的作用，所以制作叫声也是为宝可梦“赋予个性”。

**——到今天，《宝可梦》系列游戏也经历了从 Game Boy 到 3DS 的变迁。过渡到新硬件的时候，叫声也会全部重新制作吗？**

增田：有拿（Game Boy PSG 的）原始声音加工的，也有重做的。最近新登场的宝可梦就全都是新制作的，而对于旧的宝可梦，则是为了让当时就玩过的粉丝听着不觉得违和，就都是由原始叫声来加工的。从 3DS 的《X／Y》开始，宝可梦的叫声也从以往的 PSG 音，往更接近真实动物的声音转变。

**——《红／绿》是 151 种宝可梦，现在已经增加到 720 多种了。种类大幅增加，制作叫声是不是也有各种各样的困难呢？**

增田：最需要注重的一点，是要避免叫声彼此之间相似。《红／绿》的时候因为可用音色与数据容量的限制，所以会有好些叫声听起来会很像的情况。如今的硬件则是能够播放出来各种各样的音色了，也就更容易做出差异。然而，要做出既符合宝可梦形象、又介于真实与幻想之间的叫声，即使是今天也依然很费工夫。那种“好像现实里存在，但实际又没有听过”的平衡，要实现是很困难的。
