**——《红/绿》发售 20 周年了，《宝可梦》系列今后也会继续让大家喜闻乐见，在音乐方面，您有没有什么想尝试的新挑战呢？**

增田：在给 Game Boy 写声音驱动（控制 BGM 和音效播放的程序）的时代，我就想过各种点子，比如说用纯律来作曲会不会很有意思之类。现在的流行音乐多数以平均律来制作，但这样和音听起来就会比纯律的要浑浊一点。然而纯律有个缺点就是很难转调，所以当时没能实现……因此我现在也依然希望能让宝可梦的玩家们能够听听纯律的音乐。如果能够回归这个原点并向大家传递音乐的“乐趣”，那就真是太好了。

> 纯律和平均律是两种音律体系。纯律中的每个音之间频率关系都是有理数，听起来和谐悦耳，但只适合固定调性演奏。平均律是把一个八度平均分成12个相等的半音，虽然和声不如纯律纯净，但可以随意转调，但和弦的声音不如纯律的纯净。
